How Field Mapping Works

The field mapping table defines how column names translate between Square and Shopify formats. When converting a file, each Square column is matched to its Shopify equivalent using this mapping.


Adding a mapping: Use the form below the table ‚Äî enter the exact Square column name, the exact Shopify column name, and an optional note, then click + Add. The new row appears in the table. Click Save Mapping to persist it.


Removing a mapping: Click the üóë delete button on any row to remove it. The row is removed immediately from the table. Click Save Mapping to persist the change.


Special conversion rules (always applied regardless of mapping):

  • Weight: Automatically converts lb ‚Üî grams (√ó453.592)
  • Status/Archived: Shopify Status always set to Draft
  • Variants: Row structure automatically restructured between platforms
  • Unmapped fields: Silently dropped from output

Saving: Mappings are saved to your browser's local storage and persist between sessions. Use Export/Import to move mappings between devices or create backups.
